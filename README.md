# IOT Feed Service
Receives telemetry data from IOT devices

## Installation
Before you can use this repo, please run 

```npm install -g serverless```

```sls dynamodb install```

## Testing
Command for running tests is ```sls test```

## Development

To run locally
```
sls offline start
```
